## Cursor Rules — IronMebel

### Стили (SCSS / CSS)

- **Не менять файлы CSS напрямую.** Все изменения стилей делать только в SCSS (файлы `scss/*.scss`).
- После правок в SCSS запускать компиляцию (например, `sass`), чтобы обновить `css/main.css`.

### JS: защитные проверки

- **Не добавлять избыточные guard-конструкции** вида:
  - `if (typeof KeenSlider === "undefined") return;`
  - `if (!containers.length) return;`
- Вместо этого:
  - считаем, что внешние зависимости (например, `KeenSlider`) загружены там, где используются;
  - допускаем проход по пустым коллекциям (`NodeList.forEach` и т.п.) без дополнительных `if`.
